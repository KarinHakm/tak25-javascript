jQuery võrdlusülesanne

Sama funktsioon kahe erineva lähenemisega kõrvuti: tänapäevane JavaScript vs jQuery. Kummal on lühem kood?

Demo 1: Näita ja peida

Tavaline JavaScript

// Klassikaline DOM-manipulatsioon function peida() { const el = document .querySelector('#kast'); if (el.style.display === 'none') { el.style.display = 'block'; } else { el.style.display = 'none'; } }
jQuery
See tekst on nähtav ✓
// jQuery – toggle() // lühem, sisseehitatud $('#kast').toggle(); // Animatsiooniga: $('#kast').slideToggle('fast'); $('#kast').fadeToggle(300);

Demo 2: Stiili muutmine

Tavaline JavaScript

Klõpsa nuppu, et muuta stiili

// classList.toggle lisab/eemaldab document.querySelector('p') .classList.toggle('aktiivne'); // Otse stiilide muutmine: el.style.color = 'blue'; el.style.fontWeight = 'bold';
jQuery

Klõpsa nuppu, et muuta stiili

// toggleClass – sama loogika $('p').toggleClass('aktiivne'); // css() meetod muudab stiile: $('p').css({ color: 'blue', fontWeight: 'bold' });

Tavaline JavaScript
// CSS transition + style muutmine const el = document.querySelector('#kast'); el.style.transform = 'rotate(180deg)'; el.style.background = '#dc2626'; setTimeout(() => { el.style.transform = ''; el.style.background = '#2563eb'; }, 600);
jQuery
// jQuery animate() meetod $('#kast') .animate({ width: '+=40', height: '+=40' }, 400) .animate({ width: '-=40', height: '-=40' }, 400);

Kokkuvõte: millal kasutada mida?

✅ Tänapäevane JavaScript

  • Ei vaja välist teeki
  • Kiirem (vähem kood alla laadida)
  • querySelector, classList, fetch()
  • Tänapäevane standard (2015+)
  • Töötab kõikides uutes brauserites

ℹ️ jQuery

  • Vajab CDN-i või faili (~87 KB)
  • Lühem süntaks ($())
  • Sisseehitatud animatsioonid
  • Tuntud vanematel veebisaitidel
  • Kasulik, kui toetad vanu brausereid

Järeldus: Uutes projektides kasuta tänapäevast JavaScripti. jQuery-t tasub tunda, sest seda leidub paljudes vanades projektides ja WordPressi teemades.
